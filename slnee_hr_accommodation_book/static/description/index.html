Employee Accommodation Booking

In case of Business tour/Client Meeting/Personal Work employee will request to Admin Manager for accommodation booking.

Admin Manager will add accommodation & expense (company & employee contribution) details & approve employee request.

Employee contribution (expense) is directly deducted from employee payslip.

Accommodation Request

Employee will create request & submit to Admin Manager.



Approve/Refuse Request

Admin Manager will add accommodation & expense (company & employee contribution) details & approve request.



Book Accommodation & Generate Expense

Admin Manager will book accommodation for employee & generate expense.



Expense Process

Expense Manager will do expense process.



Employee Payslip Calculation

Current month expense (Employee contribution) will be deducted from employee's current month payslip.



Admin Report

Excel report (can be filter date wise) with all details.
